# MultiSig Wallet
Tutorial on building a multi-signature crypto wallet, running on Asset Chain.

## 🔨 Setup
Follow the steps below to setup the React Project.
```bash
# fork and clone the forked project
$ git clone [REPO_URL]

# Access the folder
$ cd multisig-wallet

# Install dependencies
$ yarn install

# Run app
$ yarn dev
```
